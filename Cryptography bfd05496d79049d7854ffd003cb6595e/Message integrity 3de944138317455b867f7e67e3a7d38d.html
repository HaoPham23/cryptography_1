Verification Timing Attack

def	Verify(key,	msg, sig_bytes):	
	return	HMAC(key,	msg)	==	sig_bytes

The problem is that ‘==’ implemented as a byte-by-byte comparison. It will return false when first inequality found.

Attack:

  1. Query server with random tag
  1. Loop over all possible first bytes and query server. Stop when verification takes a little longer than in step 1.
  1. Repeat for all tag bytes until valid tag found
